Some Academic Projects

Stack Implementation using C#

This project showcases a fundamental implementation of a Stack data structure in C#, built entirely from memory during a closed-book test in COIS 2020 at Trent University. Using a linked list, it supports core stack operations—push, pop, isEmpty, and display—through a menu-driven interface. Developed in Visual Studio Code, the project demonstrates proficiency in object-oriented programming, algorithmic logic, and error handling, while reinforcing key data structure concepts under time constraints.
