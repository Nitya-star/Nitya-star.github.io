# Personal Portfolio Website using HTML and CSS

This project showcases the creation of a personal portfolio website built using HTML5 and CSS3. It servers as a central platform to present academic background, projects, experiences, hobbies, and ongoing learning - combining simplicity, structure, and accessibility in design. This project was developed in Visual Studio Code, allowing real-time syntax checking.

## Overview

The Personal Portfolio Website highlights professional and personal growth through a clean, organized surface. Each page focuses on different area of development:
* **Home:** Introduction, navigation links, and contact information.
* **Studies:** Academic background and relevant coursework.
* **Projects:** Overview of completed academic projects.
* **Experiences:** All work experiences and its details.
* **Learning:** Courses currently being learnt.
* **Hobbies:** Favorite activities that inspire creativity,  focus, and a positive mindset.

## Features

1. **Structure & Accessibility**
   * Built using semantic HTML5 for logical and readable content hierarchy.
   * Unified styling through a single external main.css file.
   * Consistent navigation bar across all pages for easy browsing.
   * Accessible color contrast and WCAG Principles are followed.
2. **Design**
   * Layout created using CSS flexbox and basic box model concepts.
   * Clean, minimal design emphasizing readability.
   * Responsive formatting for both desktop and mobile screens.

## Code Structure

* **index.html:** Home page with introduction and navigation
* **studies.html:** Academic information and courses
* **projects.html:** Academic projects' descriptions and learnings
* **experiences.html:** All work experiences and its details
* **learning.html:** List of courses currently being learnt
* **hobbies.html:** Hobbies and mindset reflection
* **style.css:** External stylesheet for design and layout
* **README.md:** Project documentation with details
